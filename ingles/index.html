Ingles

1. Modos del verbo

Indicativo: se utiliza para hablar de acciones objetivas y hechos reales

Subjuntivo: se utiliza para expresar deseos, dudas, posibilidades, hipótesis y situaciones irreales

Imperativo: se utiliza para dar órdenes, hacer solicitudes o expresar deseos

2. Modos impersonales del verbo

Participio: se utiliza para formar tiempos compuestos y pasivos, como en "he comido" o "fue construido"

Gerundio: se utiliza para expresar la acción del verbo en progreso sin especificar quién la realiza, como en "estoy caminando hacia la tienda"

Infinitivo: se utiliza para expresar la acción del verbo sin especificar quién la realiza, como en "correr es bueno para la salud"

Ingles

1. Modos del verbo

Indicativo: se utiliza para hablar de acciones objetivas y hechos reales

Subjuntivo: se utiliza para expresar deseos, dudas, posibilidades, hipótesis y situaciones irreales

Imperativo: se utiliza para dar órdenes, hacer solicitudes o expresar deseos

2. Modos impersonales del verbo

Participio: se utiliza para formar tiempos compuestos y pasivos, como en "he comido" o "fue construido"

Gerundio: se utiliza para expresar la acción del verbo en progreso sin especificar quién la realiza, como en "estoy caminando hacia la tienda"

Infinitivo: se utiliza para expresar la acción del verbo sin especificar quién la realiza, como en "correr es bueno para la salud"

3. Que es el verbo

El verbo es una palabra que se utiliza para expresar una acción, un estado o una existencia del sujeto

4. Que es el sustantivo

El sustantivo es una palabra que se utiliza para nombrar a personas, animales, plantas, objetos, lugares, ideas o sentimientos

5. Que es el adjetivo

El adjetivo es una palabra que se utiliza para describir o calificar al sustantivo en la oración

6. Que diferencia hay entre predicado y complemento

La diferencia entre el predicado y el complemento es que el predicado es la parte de la oración que expresa la acción o el estado del sujeto, mientras que el complemento es una palabra o grupo de palabras que complementa o completa el significado del verbo o del sustantivo

7. Que es plural

El plural es una forma gramatical que se utiliza para indicar que hay más de una cosa o ser

8. Que es singular

El singular es una forma gramatical que se utiliza para indicar que hay una sola cosa o ser

9. Personas gramaticales

Primera persona: La primera persona gramatical se utiliza para referirse a uno mismo o al grupo al que se pertenece

Segunda persona: La segunda persona gramatical es una categoría gramatical que nos permite dirigirnos directamente a la persona o personas a las que nos estamos comunicando

Tercera persona: La tercera persona gramatical es una categoría gramatical que nos permite referirnos a alguien o algo que no es el hablante o el receptor del mensaje

19. Que es el adverbio

El adverbio es una palabra invariable que tiene la función de modificar o complementar un verbo, un adjetivo u otro adverbio

20. Que es el superlativo

El superlativo es una forma gramatical que se utiliza para expresar el grado máximo de una cualidad o característica. En español, el superlativo se forma añadiendo el sufijo "-ísimo/-ísima" a la raíz del adjetivo en grado positivo. Por ejemplo, "alto" en grado positivo se convierte en "altísimo" en grado superlativo. El superlativo se utiliza para destacar la cualidad o característica de un objeto o persona en comparación con otros objetos o personas del mismo grupo. En resumen, el superlativo es una forma gramatical que nos permite expresar el grado máximo de una cualidad o característica.

21. Condicional cero

El condicional cero es una estructura gramatical que se utiliza para expresar una acción que se realizará si se cumple una condición en el presente

La estructura del condicional cero es: "Si + presente simple, presente simple"

Puede utilizar "if" o "when"

El condicional if puede ir antes o despues de la consecuencia

Siempre se usa "," para separar la condicion de la causa

No se usa "," si la causa va antes de la condicion

Estructura: if + sujeto + verbo en presente simple + causa + "," + sujeto + verbo en presente simple + consecuencia
